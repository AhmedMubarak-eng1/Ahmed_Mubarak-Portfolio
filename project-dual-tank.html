Project Overview

System Architecture

This project involved the full development and programming of a dual-tank pump control system on an Allen-Bradley CompactLogix PLC emulated via Studio 5000. The system controls water levels across two interconnected tanks using four pumps (P1–P4) and four valves (V0–V3), with the PLC making autonomous decisions to maintain safe operating levels.

A core design goal was reusability and standardization: all pump and valve logic was encapsulated in Add-On Instructions (AOIs) — meaning each device shares the same proven logic block, parameterized for its specific tag. This drastically reduces programming time, eliminates copy-paste bugs, and simplifies future maintenance.

The FactoryTalk View HMI provides operators with a live graphical overview of the system, mode controls, simulation tools, and fault management — everything needed to run, test, and maintain the system from a single interface.

PLC Control Logic

What the PLC Does

The PLC program is structured entirely around Add-On Instructions (AOIs) — reusable logic blocks defined once and instantiated for each device. This approach eliminates redundancy, ensures consistent behaviour, and makes the codebase dramatically easier to maintain and extend.

Pump Alternation Logic

Balances runtime across all pumps by rotating lead/lag assignments, preventing premature wear on any single unit.

Maintenance Mode

Any pump or valve can be taken offline for service while the system continues to operate with the remaining assets — no shutdowns required.

Fault Handling & Interlocks

Built-in fault detection with configurable fault ignore, preventing unsafe operation and protecting equipment from harmful conditions.

AOI-Based Architecture

All pump and valve logic encapsulated in Add-On Instructions — instantiated with one block per device, keeping the program clean and scalable.

HMI Features

FactoryTalk View Operator Interface

The FactoryTalk View ME application gives operators complete situational awareness and control. The interface is designed around three layers: overview (see the whole system at a glance), mode control (switch any device between Auto/Manual/Maintenance), and alarm & fault management.

Live Tank Level Display Real-time numeric display of Tank 1 and Tank 2 levels, with animated graphical tank representations showing current fill state.
Per-Device Mode Selection Popup screens for Pumps Mood Select and Valves Mood Select — operators can independently set each device to Auto, Manual, or Maintenance from a single screen.
Alarm Annunciation Dedicated alarm indicators for high/low tank levels (e.g. T2 LOW LEVEL shown in red), pump faults, valve faults, and abnormal operating states — clearly visible from the main overview.
Manual Start / Stop / Maintenance Controls Each pump and valve panel includes Manual Start, Manual Stop, Auto Stop, Maint. Start, and status indicators (Pump Ready, Valve Ready) for full operator override capability.
Simulation Mode Dedicated Simulation and Sim. Inputs buttons allow the system to be tested in a simulated environment — enabling commissioning and training without live equipment.
Fault Ignore Capability Individual P1–P4 and V0–V3 Ignore Fault toggles allow maintenance personnel to suppress specific faults during commissioning or repair activities.
Key Outcomes

What Was Delivered

Fully functional dual-tank level control with automatic pump & valve sequencing PLC autonomously maintains tank levels within set ranges by starting and stopping pumps based on level feedback.
Reusable AOI-based program structure for pumps and valves Single Add-On Instruction definition instantiated per device — consistent behaviour, easier debugging, and zero code duplication.
Zero-downtime maintenance mode with pump alternation Any pump can be taken offline for service without stopping the system, and runtime is evenly distributed to extend equipment life.
Professional-grade HMI with alarms, simulation, and per-device control Operators have full visibility and control from a clean, logically organized interface with color-coded status indicators.
