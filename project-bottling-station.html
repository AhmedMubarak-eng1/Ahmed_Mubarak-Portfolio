Industrial Process Control

SMC IPC-202A Bottling Station – Omron PLC & HMI

McMaster University – Lab Project Sept – Dec 2025 Team of 3

Overview

Working in a team of three, designed and commissioned the IPC-202A Bottling Station (Gravity Feeder) — a subsystem of the SMC IPC-200 Industrial Process Control training system at McMaster University. The project involved installing, wiring, and programming a complete automated bottling process that simulates real industrial fill-and-cap production lines used in food and beverage manufacturing.

The station automates the full bottle lifecycle: feeding bottles via gravity, filling with liquid, placing lids, and transferring bottles downstream — all controlled through an Omron PLC with a custom-built operator HMI. The project provided hands-on experience with PLC programming, pneumatic systems, safety interlock design, alarm logic, and inter-station communication.

IPC-200 Table 5 — McMaster University Automation Lab. Omron HMI (left), pneumatic fill & cap actuators (centre), wired control panel (rear).

Project Progress

Completed wiring of all I/O devices — proximity sensors, solenoid valves, pneumatic actuators, and emergency stop circuits wired to the control panel and verified against the I/O list
Mapped and tested every I/O point in the Omron PLC, confirming accurate sensor feedback, correct actuator response, and signal integrity across all channels
Integrated emergency stops and safety interlocks in compliance with safety standards, preventing unsafe actuator sequences and protecting operators during manual operation
Programmed automatic and manual control modes in Omron Sysmac Studio — automatic mode runs the full detect → fill → cap → transfer sequence; manual mode allows step-by-step operation for setup and troubleshooting
Implemented alarm logic for fault detection: empty feeder detection, low/high liquid level faults, cycle timeout alarms, sensor failures, and actuator fault conditions
Debugged pneumatic timing to synchronize the gravity feeder, filling cylinder, lid placement actuator, and indexing plate movement into a smooth, reliable sequence
Developed complete HMI screens in Omron HMI software — operator screens include system status overview, alarm display with fault descriptions, mode selection (Auto/Manual), production counter, and manual override controls
Performed full system testing and validation — all sequences, interlocks, alarms, and HMI functions verified against design specifications with zero interlock failures across all test cycles
Implementing inter-station communication with IPC-201 (liquid preparation unit) for coordinated material supply across the full IPC-200 multi-station system
Conducting full multi-station integration tests with other IPC-200 modules to validate coordinated operation across the complete process line

What Was Built

  • Installed and wired all field devices — proximity sensors, solenoid valves, pneumatic actuators, and E-stop circuits — to the control panel, following the system schematic and I/O list
  • Tested every I/O channel individually to verify signal accuracy, wiring integrity, and correct device response before any programming began
  • Programmed the automatic bottling sequence in Sysmac Studio: bottle detect → gravity feeder release → fill cylinder extend → fill → fill cylinder retract → lid placement → indexing plate transfer
  • Programmed a manual mode allowing individual actuator control for setup, calibration, and fault troubleshooting without running the full sequence
  • Designed safety interlocks that prevent actuator conflicts — e.g., fill cylinder cannot extend unless bottle-present sensor is confirmed; indexing plate cannot advance while any actuator is mid-stroke
  • Built alarm logic covering empty feeder, liquid level out-of-range, cycle timeout, sensor fail, and actuator fault conditions — all surfaced to the HMI with descriptive messages
  • Designed and deployed HMI operator screens: main status overview, alarm/fault page, mode selector, bottle counter, and individual manual control buttons
  • Tuned pneumatic cylinder timing in software to eliminate bounce and misfeed at the gravity feeder and ensure consistent lid placement across all test cycles
  • Validated the complete system against specifications — all sequences, interlocks, alarms, and HMI functions passed with no interlock failures

Technical Highlights

PLC Platform

Omron NX/NJ-series PLC programmed in Sysmac Studio using Ladder Diagram (LD) and Structured Text (ST) for sequence control, interlock logic, and alarm management.

HMI Development

Full operator interface built in Omron HMI software — real-time status, alarm banners with fault codes, Auto/Manual mode toggle, production counters, and manual actuator override screens.

Pneumatic Systems

Coordinated four pneumatic actuators (feeder release, fill cylinder, lid placement, indexing plate) with tuned timing to achieve smooth, repeatable bottle handling across all operating cycles.

Safety & Interlocks

Hardwired E-stop circuits and software interlocks prevent unsafe actuator sequences. All safety logic was tested with forced fault conditions to verify correct system response under failure modes.

Results & Impact

Delivered a fully functional, dual-mode industrial bottling station with complete I/O commissioning, safety interlock logic, alarm management, and a professional operator HMI — all validated through structured system testing. The station ran reliably through all test cycles with zero interlock failures, and the HMI provided operators with clear visibility into system state, active alarms, and production counts. The project directly demonstrated team-based industrial installation, PLC programming, pneumatic system integration, and operator interface design at a production-grade standard.
