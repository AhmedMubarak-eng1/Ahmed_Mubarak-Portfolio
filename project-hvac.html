Building Automation

HVAC Control System – Schneider M172 PLC

McMaster University – Learning Project Sept – Dec 2025

Overview

Designed and built a small-scale HVAC automation system using the Schneider M172 programmable controller — replicating real building automation functions including zone-based heating and cooling control, operational scheduling, alarm management, and BACnet network integration.

What Was Built

  • Designed and wired a professional-grade control panel including I/O expansion modules, an industrial Ethernet switch, and a motorized water valve actuator
  • Programmed the Schneider M172 PLC with sequences for heating and cooling loops, setpoint control, and time-based operational scheduling
  • Implemented alarm logic for out-of-range temperature conditions, valve fault detection, and communication loss
  • Configured BACnet communication protocol for integration with a building management network
  • Developed complete technical documentation including logic flow diagrams, panel wiring schematics, and system integration guide
  • Tested all control sequences, alarm conditions, and scheduling functions to verify correct operation

Results & Impact

Produced a fully documented, operational HVAC control system demonstrating real-world building automation skills — from panel design and wiring through PLC programming, network protocol configuration (BACnet), and complete technical documentation. The project replicated the engineering workflow used in real commercial building automation projects.
